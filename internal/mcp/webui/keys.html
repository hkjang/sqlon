MCP 키란?
MCP 클라이언트(Claude, qwen-code 등)가 이 서버의 /mcp HTTP 엔드포인트에 접속할 때 쓰는 API 키입니다. 클라이언트 설정에 Authorization: Bearer jsk_... 또는 X-MCP-Key: jsk_... 헤더로 넣으세요. 키는 발급 순간 한 번만 표시되며 서버에는 해시만 저장됩니다. 유출이 의심되면 즉시 [회전] 또는 [폐기]하세요.
